# F2V Golden Path

## Deterministic Sequence
1. Open the Flow workspace
2. Select `Video`
3. Select `Frames`
4. Set `9:16`
5. Set `1x`
6. Verify Veo `3.1 Lite`
7. Verify Start slot visibility
8. Attach Start frame
9. Verify Start preview
10. Insert prompt
11. Stop before Generate unless explicitly authorized

## Current Freeze
- Steps 2 to 6 are treated as the proven configuration path.
- Steps 8 and 9 are unstable and belong to the rebuild lane.
